C++中使用Box-Muller公式产生满足正态分布 $ N(\mu, \sigma^2) $ 的随机数：
$$ X = \mu + \sigma cos(2\pi U_1)\sqrt{-2\ln U_2} $$
其中 $ U_1 $，$ U_2 $是$ [0, 1] $内均匀分布的随机数。
